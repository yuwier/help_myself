Если у вас нет Python 11.0 версии, приложение не запустится.

---

Для проверки необходимой сессии - найдите папку с ее номером в папке sessions
sessions / session_1 | session_2 | session_3 | session_4 
В каждой из них вы найдете readme.txt

---

Для запуска приложения - найдите в корневой папке (в которой находитесь), файл LaboratoryApp.exe, запустите его.
